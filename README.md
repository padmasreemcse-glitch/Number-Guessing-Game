# 🎯 Number Guessing Game

A simple Python terminal game where the player guesses a randomly generated number between 1 and 100.

## Features
- Random number generation
- Unlimited attempts
- Too High / Too Low hints
- Attempt counter
- Interactive terminal interface with emojis

## Technologies
- Python 3
- random module

## How to Run

```bash
python number_guessing_game.py
```
